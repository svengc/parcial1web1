# 📘 Parcial Web 1 – Lógica de Programación con JavaScript

## 📖 Descripción
Este repositorio contiene la solución de los ejercicios del **Primer Parcial de la asignatura Web 1**.

El objetivo del proyecto es demostrar el uso de **lógica de programación en JavaScript**, aplicando estructuras básicas de programación para procesar información ingresada por el usuario y mostrar resultados en consola.

Durante el desarrollo se aplicaron conceptos fundamentales como:

- Variables
- Conversión de datos (`parseFloat`, `parseInt`)
- Validación de datos con `isNaN`
- Estructuras de control (`if`, `while`)
- Interacción con el usuario mediante `prompt`
- Salida de información con `console.log`

Los ejercicios simulan diferentes situaciones donde el sistema recibe datos del usuario, los procesa y genera resultados.

## 🧠 Arquitectura del Proyecto

El proyecto está organizado utilizando una estructura basada en **MVC (Model - Controller - View)** para separar responsabilidades dentro del código.

| Componente | Función |
|---|---|
| **View** | Archivos HTML encargados de la interfaz |
| **Controller** | Contiene la lógica y funciones del programa |
| **Model** | Maneja los datos y el flujo del programa |

Esta organización permite mantener el código **ordenado, modular y fácil de mantener**.
