# Parcial Web 1 – Lógica de Programación con JavaScript

## Descripción
Este repositorio contiene la solución de los ejercicios del primer parcial de la asignatura Web 1.  
El objetivo del proyecto es demostrar el uso de lógica de programación en JavaScript utilizando:

- Variables
- Conversión de datos (parseFloat, parseInt)
- Validación de datos con isNaN
- Estructuras de control (if, while)
- Interacción con el usuario mediante prompt y console.log

Los ejercicios simulan diferentes situaciones donde se deben procesar datos ingresados por el usuario y mostrar resultados en consola.
